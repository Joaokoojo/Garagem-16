# Garagem 16
O carro não liga. São muitos problema para você resolver, e o mecânico não dá trégua. Bem-vindo à **Garagem 16**.
---
O Garagem 16 é um jogo de desafio de tempo onde a precisão e a velocidade são cruciais. Como o mecânico responsável, o seu objetivo é 
manter o veículo operacional enquanto problemas mecânicos surgem de forma aleatória e em ritmo acelerado. A pergunta não é 
se você consegue consertar, mas quantos problemas você consegue resolver antes que o tempo esgote.

## O Objetivo
Você precisa realizar a maior quantidade de reparos possíveis no menor espaço de tempo.

## Mecânicas de Jogo
* **Gestão de Crise:** Problemas surgem dinamicamente e exigem diagnósticos rápidos.
* **Priorização:** Nem todo problema é simples; decida o que consertar primeiro para manter o carro rodando.
* **Time Attack:** O jogo testa sua capacidade de manter a calma sob pressão enquanto o cronômetro corre.

## Tecnologias Utilizadas
* Feito em Godot
* Linguagens: GDScript e C#
* Outras ferramentas: Aseprite
